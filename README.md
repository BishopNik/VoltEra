# VoltEra

Концепт преміальної енергетичної платформи українською мовою.

## Локальний запуск

```bash
python3 -m http.server 8787
```

Відкрити: `http://127.0.0.1:8787/`

Адмін-панель: `http://127.0.0.1:8787/admin.html`

## Що реалізовано

- адаптивна головна сторінка з light/dark темою;
- жива схема енергосистеми та симулятор відключення мережі;
- калькулятор навантаження й автономності;
- інтерактивний підбір системи, фільтри комплектів, відгуки та FAQ;
- форма консультації з готовим success-станом;
- три SEO-посадкові сторінки;
- порівняння інверторів Deye, Anenji та Easun;
- галерея об'єктів із інтерактивними паспортами систем;
- відгуки, які можна додавати й гортати, з відповідями VoltEra;
- журнал корисних матеріалів і мініфорум «Енергоколо»;
- адаптивна адмін-панель для заявок, відгуків, проєктів, статей та обладнання;
- JSON-LD, Open Graph, canonical, robots.txt, sitemap.xml та web manifest;
- доступна семантична розмітка й підтримка `prefers-reduced-motion`.

Перед публікацією замініть демонстраційні телефон, email і домен на фактичні та підключіть форму до CRM/API.

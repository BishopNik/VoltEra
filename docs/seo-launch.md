# Запуск SEO для voltares.pp.ua

## Що вже зроблено в коді

- канонічний домен `https://voltares.pp.ua/`;
- `robots.txt` дозволяє індексацію публічних сторінок і посилається на sitemap;
- sitemap містить головну, каталог, посадкові сторінки брендів, рішення, галерею,
  Енергоколо та опубліковані статті;
- окрема посадкова сторінка `/obladnannia/deye.html` оптимізована під запити
  «інвертори Deye в Україні», «купити інвертор Deye», підбір і монтаж;
- додані canonical, унікальні title/description, внутрішні посилання,
  BreadcrumbList, Service, LocalBusiness та FAQPage;
- незавершена англійська копія не створює дублікати українських сторінок.

## Обов'язкові дії після deployment

1. Додати `voltares.pp.ua` як Domain property у Google Search Console і пройти
   перевірку DNS TXT-записом.
2. Надіслати `https://voltares.pp.ua/sitemap.xml` у розділі Sitemaps.
3. Через URL Inspection перевірити й запросити індексацію щонайменше для:
   - `https://voltares.pp.ua/`;
   - `https://voltares.pp.ua/catalog.html`;
   - `https://voltares.pp.ua/obladnannia/deye.html`.
4. Створити або оновити Google Business Profile: однакові назва Voltares / ІНК,
   адреса, телефони та посилання на сайт.
5. Отримати реальні посилання на Deye-сторінку від постачальників, партнерів,
   локальних каталогів і сторінок виконаних об'єктів.
6. Публікувати в журналі корисні матеріали з реальними фото, вимірами й моделями
   обладнання; кожна стаття повинна мати власну тему й відповідати на конкретний
   пошуковий запит.

## Контроль результату

Щотижня перевіряти у Search Console:

- Pages / Indexing — причини невключення сторінок до індексу;
- Performance — покази, позиції та CTR для `voltares`, `deye`, `інвертор`;
- Core Web Vitals — мобільні й desktop URL;
- Links — нові зовнішні та внутрішні посилання.

Позиція в першій десятці не гарантується метатегами або кількістю ключових слів.
Вона залежить від індексації, конкуренції, корисності контенту, довіри до домену,
локальних сигналів, посилань і поведінки користувачів.

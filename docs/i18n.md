# Мова сайту Voltares

Публічна версія сайту працює українською мовою. Це свідоме обмеження: усі записи
MongoDB (товари, статті, FAQ, питання, відповіді, відгуки й об'єкти) мають бути
однією мовою з оболонкою сторінки.

Старий параметр `?lang=en` видаляється з URL, а `en.html` переводить відвідувача
на канонічну українську сторінку. Англійські URL не додаються до sitemap і не
позначаються через `hreflang`.

## Як правильно повернути англійську версію

1. Створити окремі URL у каталозі `/en/`, а не параметр на українській сторінці.
2. Додати в CRM переклад кожного динамічного поля (`title`, `answer`, `body`,
   `description`, `excerpt`) і не показувати український fallback на англійській сторінці.
3. Додати взаємні `hreflang="uk-UA"` / `hreflang="en"`, окремі canonical, title,
   description і структуровані дані.
4. Лише після повного перекладу додати англійські URL до sitemap.

Словники старої експериментальної оболонки залишені в `i18n.js` як база для
майбутньої міграції, але перемикач зараз вимкнений.

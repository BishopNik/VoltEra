# Багатомовність сайту ІНК

Сайт має одну основну сторінку `index.html`. Українська мова є базовою, інші мови вмикаються через параметр:

- українська: `/`
- англійська: `/?lang=en`

`en.html` залишено тільки як сумісний вхід для старих посилань і одразу переводить на `/?lang=en`.

## Де переклади

Усі статичні переклади лежать у `i18n.js`:

- `LANGUAGES` — список доступних мов;
- `I18N_MESSAGES` — словники перекладів;
- `I18N_BINDINGS` — схема, який ключ перекладу застосовується до якого DOM-елемента.

## Як додати нову мову

1. Додати мову в `LANGUAGES`, наприклад:

   ```js
   pl: { label: 'PL', htmlLang: 'pl' }
   ```

2. Додати `pl` в `I18N_MESSAGES` з такими самими ключами, як `uk` та `en`.
3. Перемикач мов зʼявиться автоматично.

## Динамічний контент із MongoDB

Публічні записи з API (`reviews`, `projects`, `articles`, `questions`) зараз зберігаються українською. Для мов, відмінних від `uk`, сайт не підміняє статичні блоки українськими API-даними.

Наступний правильний крок для повної багатомовної CRM:

```json
{
  "title": { "uk": "Назва", "en": "Title" },
  "excerpt": { "uk": "Опис", "en": "Description" }
}
```

Тоді API зможе віддавати локалізоване поле за поточною мовою.

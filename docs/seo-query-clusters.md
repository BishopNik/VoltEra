# Кластери пошукових запитів Voltares

Джерело даних — експорт **Google Search Console → Performance**. Не об’єднувати різні наміри лише через схожі слова: запит на купівлю, налаштування та помилку обладнання потребує різних сторінок.

| Запит | Мова | Країна | Impressions | Clicks | CTR | Average position | Цільова сторінка | Намір | Дія | Статус контенту |
|---|---|---|---:|---:|---:|---:|---|---|---|---|
| _вставити з GSC_ | uk | UA | 0 | 0 | 0% | 0 | `/` | commercial / informational / support | create / improve / merge / redirect | backlog |

## Правила статусів

- `backlog` — кластер перевірений, робота не почалася;
- `brief` — є структура матеріалу та цільова сторінка;
- `writing` — готується експертний текст;
- `review` — технічна й фактологічна перевірка;
- `published` — сторінка опублікована та додана у внутрішні посилання;
- `measure` — минуло 4–8 тижнів, аналізуються покази, CTR і позиція.
